# 中文

## MetalPlayer

MetalPlayer 是一款面向 macOS Apple Silicon 的实验性高性能视频播放器，专注于动画播放、实时补帧、超分辨率、硬件解码和高质量渲染。

当前版本仍处于 **Beta 测试阶段**，欢迎反馈问题和建议。

## 主要特性

- 支持 macOS Apple Silicon
- 支持 MKV / MP4 等常见视频格式
- 支持 RIFE 实时补帧
- 支持 MetalFX 超分辨率
- 支持外挂字幕和内嵌字幕

## 系统要求

- macOS 14 或更新版本推荐
- Apple Silicon Mac 推荐
  - M1 / M2 / M3 / M4 / M5 系列
- 建议使用较新的 macOS 版本以获得更好的 VideoToolbox、Metal 和 Core ML 支持

# English

## MetalPlayer

MetalPlayer is an experimental high-performance video player for macOS Apple Silicon, focused on anime playback, real-time frame interpolation, upscaling, hardware decoding, and high-quality rendering.

The current version is still in **Beta**, and feedback is welcome.

## Features

- Supports macOS Apple Silicon
- Supports common video formats such as MKV and MP4
- Supports real-time RIFE frame interpolation
- Supports MetalFX upscaling
- Supports external and embedded subtitles

## System Requirements

- macOS 14 or later is recommended
- Apple Silicon Mac is recommended
  - M1 / M2 / M3 / M4 / M5 series
- A recent macOS version is recommended for better VideoToolbox, Metal, and Core ML support
